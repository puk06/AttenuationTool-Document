# 防音室の追加

ここでは、２種類の防音室の追加方法について解説します。


## 基本的な防音室の追加方法

1. Unityのヒエラルキー上で右クリック
2. `防音室ギミック/防音室/防音室を作成` から

- 部屋がシンプルな直方体（箱）形状の場合は `Box Collider`
- 部屋が球体形状の場合は `Sphere Collider`
- 部屋がカプセル形状（円柱 + 半球）の場合は `Capsule Collider`

3. 上から追加したい部屋の形状に合うものをクリック

## カスタムシェーダーが適用された防音室の追加方法

:::note
カスタムシェーダーが適用された防音室は、デフォルトのコライダーの種類が `Box Collider` となっています。
:::

1. Unityのヒエラルキー上で右クリック
2. `防音室ギミック/防音室/防音室を新規作成(マテリアル付き)` をクリック
