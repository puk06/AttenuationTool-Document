# 調整スライダーの追加

ここでは、防音室の防音率を調整するスライダーの追加方法について解説します。

## スライダーオブジェクトの追加方法

1. Unityのヒエラルキー上で右クリック
2. `防音室ギミック/調整スライダー/防音率調整スライダーの設置` をクリック
3. 位置や大きさなどを調整したり、不要なもの（背景など）を削除する

### 特定の防音室の防音率を変更する

:::info
これは、カスタム防音率(`Custom Attenuation Rate`)が適用された防音室に対応しています。
:::

1. 追加されたスライダーオブジェクトをクリック
2. インスペクター内から `Attenuation Board` コンポーネントを開く
2. カスタム防音率が適用されたオブジェクト(`Custom Attenuation Rate`)を上から２番目の欄にD&Dして適用する。

### ローカルに変更する

:::info
他の人に聞かれたくないなどの用途には使用できなくなりますが、自分だけ周りの音を静かにしたいなどの用途に使えます。
:::

1. 防音率調整スライダーの `ローカルモード` にチェックを入れるだけで機能します。
