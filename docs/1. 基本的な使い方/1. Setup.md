# セットアップ

ここでは、防音室ギミックのセットアップ方法について解説します。

:::note
ここでは防音室の追加方法については解説していません。追加方法に関しては、左のパネルの `防音室の追加` を参照してください。
:::

### 手順

1. Unityのヒエラルキー上で右クリック
2. `防音室ギミック/セットアップ/セットアップ(通常)` をクリック
3. 追加された `Attenuation Tool` オブジェクトのインスペクターを開く
4. `Attenuation Tool` コンポーネント内の値を調整する

:::info
全てのコンポーネントの値それぞれに説明が書かれてあります。

値のタイトル(例: `防音の強度`)の下の英語の名前(例: `Attenuation Rate`)をクリックし、カーソルを当て続けるとその値の説明が出ます！

コンポーネント内の値を調整する際にご活用ください！
:::

:::warning
Prefabを手動で設置したり、防音室の追加を手動でしたりするのはお控えください。

AttenuationToolコンポーネントは通常、ワールドに1つのみ設置できます（防音室は何個でも設置可能です！）

何をしているのか分かる人のみPrefabから手動で追加してください。
:::
