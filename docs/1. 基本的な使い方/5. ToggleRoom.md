# 防音室のON / OFFを切り替える

ここでは、防音室のON / OFFを切り替える方法について解説します。

:::info
v1.8.4にはトグルが機能しないというバグがあります。

v1.8.5以降のバージョンで修正済みですので、もしv1.8.4をお使いであればアップデートしていただけると治るかと思われます。（v1.8.3以前であればトグルは機能します）
::

## 手順
ただ **コライダーの入ったオブジェクトをボタンなどでON/OFFするだけ** で良いです。

Luraさんや、よどころさんのオブジェクトトグルスイッチなどでコライダーが入ったオブジェクトをON / OFFすることで切り替えることも出来ますし、CanvasのToggleで作ることも可能です。

:::info
私は**Lura**さんの **Lura's Switch** をおすすめします。

商品リンク: https://booth.pm/ja/items/1969082
:::
