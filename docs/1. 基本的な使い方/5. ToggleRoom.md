# 防音室のON / OFFを切り替える

ここでは、防音室のON / OFFを切り替える方法について解説します。

## 手順
ただ **コライダーの入ったオブジェクトをボタンなどでON/OFFするだけ** で良いです。

Luraさんや、よどころさんのオブジェクトトグルスイッチなどでコライダーが入ったオブジェクトをON / OFFすることで切り替えることも出来ますし、CanvasのToggleで作ることも可能です。
