# 防音室エリアの結合（同期）

ここでは、複数の防音室を「同一の防音室（同じ音響環境）」として扱うための設定方法について解説します。

## 手順

1. ヒエラルキー内の設定したい防音室オブジェクト上で右クリック
2. `防音室ギミック/防音室のカスタム/防音室の同期` をクリック
3. 追加された `Attenuation Sync Collider Index` コンポーネント内の `同期先のコライダー` 欄に、基準となる防音室を設定する

この操作により、現在選択しているオブジェクトの内部IDが、指定した「同期先」と統一されます。

:::info
**設定時の注意点**
* **設定の継承**: 同期先のコライダーが設定されると、その防音室自身に付いている個別のカスタマイズ設定は無効化され、すべて**同期先のコライダーの設定**が引き継がれます。
* **拡張用コライダー**: このコンポーネントが付いたオブジェクトは、完全に「同期先の防音範囲を広げるための拡張コライダー」として扱われる点に注意してください。
:::
