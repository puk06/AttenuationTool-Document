# 防音室の大きさの変更

ここでは、２種類の防音室の大きさの変更方法について解説します。

:::warning
コライダーオブジェクトの回転軸（Rotation）を変更すると、見た目どおりに動作しないことがあります。

これは `Collider` が内部的に使っている `Bounds` の仕様が原因です。

`Collider` はちゃんと回転しているように見えても、 `Bounds`は回転せずにその範囲全てを覆う形の範囲に変形します。

以下は Unityの公式ドキュメントの `Bounds` の仕様と、その日本語訳です
> An axis-aligned bounding box, or AABB for short, is a box aligned with coordinate axes and f**ully enclosing some object**. Because the box is **never rotated with respect to the axes**, it can be defined by just its center and extents, or alternatively by min and max points.

> 翻訳: 軸並行バウンディングボックス、略してAABB（Axis-Aligned Bounding Box）は、座標軸に並行（軸に揃っている）な、**あるオブジェクトを完全に囲むボックス**です。このボックスは**軸に対して回転することが決してないため**、中心点と範囲（エクステント）だけで定義できます。あるいは、最小点（min point）と最大点（max point）によって定義することも可能です。

引用: https://docs.unity3d.com/ScriptReference/Bounds.html
:::

## Box Colliderの大きさを調整する

:::info
こちらの方が細かい調整が出来ておすすめです！
:::

1. 大きさを変更したい防音室のオブジェクトをクリック
2. インスペクター内の `Box Collider` コンポーネントを開く
3. コンポーネントメニュー内の一番上の `Edit Collider` をクリック
4. シーンビューの中のBox Colliderのそれぞれの面の中心に点が出るため、その点をマウスで操作して大きさを変更する

## Unityのオブジェクトのスケールなどで調整する

1. Unityのヒエラルキー上で右クリック
2. `R`キーや、`T`キーを押し、サイズを変更する
