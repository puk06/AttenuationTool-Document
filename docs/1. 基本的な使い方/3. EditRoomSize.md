# 防音室の大きさの変更

ここでは、２種類の防音室の大きさの変更方法について解説します。

:::warning
コライダーオブジェクトの回転軸（Rotation）を変更すると、見た目どおりに動作しないことがあります。

これは `Collider` が内部的に使っている `Bounds` の仕様が原因です。

Bounds は「**オブジェクトをすっぽり囲む箱**」ですが、
オブジェクトが回転しても、この箱自体は回転しません。

以下はUnityのドキュメントからの引用です。
> An axis-aligned bounding box, or AABB for short, is a box aligned with coordinate axes and fully enclosing some object. Because the box is **never rotated with respect to the axes**, it can be defined by just its center and extents, or alternatively by min and max points. (引用: https://docs.unity3d.com/ScriptReference/Bounds.html)
:::

## Box Colliderの大きさを調整する

:::info
こちらの方が細かい調整が出来ておすすめです！
:::

1. 大きさを変更したい防音室のオブジェクトをクリック
2. インスペクター内の `Box Collider` コンポーネントを開く
3. コンポーネントメニュー内の一番上の `Edit Collider` をクリック
4. シーンビューの中のBox Colliderのそれぞれの面の中心に点が出るため、その点をマウスで操作して大きさを変更する

## Unityのオブジェクトのスケールなどで調整する

1. Unityのヒエラルキー上で右クリック
2. `R`キーや、`T`キーを押し、サイズを変更する
