# カスタム防音率

ここでは、特定の防音室だけ防音率を変える方法について解説します。

## 手順

1. ヒエラルキー内の適用したい防音室オブジェクト上で右クリック
2. `防音室ギミック/防音室のカスタム/カスタムの防音率を適用` をクリック
3. 追加された `Custom Atenuation Rate` コンポーネント内の値を調整する

:::tip
このコンポーネントが適用されているコライダーは、外から中の声を聞く時の防音率を別で設定することが可能です。

防音室の中から聞こえる外の声は通常の防音率が適用され、外から聞く防音室の中の声はまた違う防音率が適用され、もっと小さく、もしくは大きく（スピーカーのような感じ）などの使い方ができます！

詳細は `Custom Atenuation Rate` コンポーネント内の `外からは別の防音率を適用する` と `外から聞こえる時の防音の強度` の欄をチェックしてください！！
:::
