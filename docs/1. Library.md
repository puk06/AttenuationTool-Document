# 依存ライブラリについて

現在、このギミックは私のU#ライブラリである「UdonUtils」に依存しています。
このリポジトリはVPM経由でダウンロードすることが可能です！

現在、UdonUtilsは、VPM経由でのダウンロードのみ対応しています。

## ダウンロード

フォルダ内にあるショートカットを使用して追加するか
下のリンクからVCC、ALCOMへリポジトリを追加してください。

**追加リダイレクトリンク**: https://vpm.pukosrv.net/addRepo

**ダイレクトリンク(vpm.json)**: https://vpm.pukosrv.net/

追加後、プロジェクトの管理ボタンからUdonUtilsを追加することで、そのプロジェクトでライブラリが使用可能になります。

## アップデートについて

アップデートもVPM経由で行われるため、定期的に確認してもらえると嬉しいです。
