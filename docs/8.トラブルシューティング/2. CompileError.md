# コンパイルエラーが出る...

おそらく、依存ライブラリがプロジェクトに入っていません。

## 解決手順

1. 左のパネルの `依存ライブラリ` を開き、ダウンロード + インポートを行ってください。

:::info
もしこれで解決しない場合は、他のギミックがコンパイルエラーを起こしている可能性が高いです。

エラー文を開いた時に出る文章で `(at [フォルダ]/[ファイル名].cs:[ファイルのエラーを起こしている行])` というのでどこがエラーを起こしているか分かるため、そのフォルダやパス名などからどのギミックが悪さしているかを確認できます！

もしここに `Attenuation Tool` が含まれていた場合は、Boothのメッセージ機能を使って報告してもらえると嬉しいです。個別で対応します！
:::
