# 声を逆に大きくする (拡声)

ここでは、防音率を使って逆に声を大きくする方法について解説します。

### 仕組み

防音率というのは **音を小さくする倍率** の事を表します。

ですがこれを負の値にすると逆に **音を大きくする倍率** になります。

これを使用した機能になります。

### ワールド全体に声を届かせる

1. 防音率を `-99999` などの大きな負の数に設定する。

:::warning
これを全体で適用してしまうと、コライダーの中から聞こえる外の声も一緒に大きくなってしまいます。

`Custom Attenuation Rate` コンポーネント内の `外からは別の防音率を適用する` + `外から聞こえる時の防音の強度` のみ変更することをおすすめします。
:::
