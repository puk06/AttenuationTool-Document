# 音を籠もらせる

ここでは、Audio Low Pass Filterを使ってAudio Sourceの音を籠もらせる方法について解説します。

:::info
`Attenuation Audio Source` `Attenuation Multiple Audio Sources` コンポーネント内の `ミュートの代わりに音量で制御する` をオンにしていないと機能しません。（完全にミュートだと籠もらせる音がないため）
:::

## 手順

:::warning
動画プレイヤーなどの特殊なAudio Sourceには対応していません
:::


1. ヒエラルキー内の追加したいAudio Sourceを含んでいるオブジェクト上で右クリック
2. `防音室ギミック/Audio Source/Audio Low Pass Filterで音を籠もらせる` をクリック
